## Dependencies
To use this project, you will need to install the following dependencies:

- [CogLink](https://github.com/PerformanC/CogLink)
- [Concord](https://github.com/Cogmasters/Concord)

Follow the installation instructions provided in the CogLink repository to install the dependency.

## Installation

To install and use this project, follow these steps:

1. Clone the repository:

    ```bash
    git clone https://github.com/m1st-gh/Lyrical.git
    ```

2. Navigate to the project directory:

    ```bash
    cd Lyrical
    ```

3. Build the project:

    ```bash
    make
    ```

## Usage

To use this project, follow these steps:

1. Configure the config file in `/Config` and move it to the project directory:

    ```bash
    cd Lyrical
    ```

2. Run Lyrical:

    ```bash
    ./out/Lyrical
    ```
